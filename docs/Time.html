NAME

 Log2t::Time - A library that assists with time manipulation (part of log2timeline)

METHODS

Win2Unix( low, high )

This function, written by Andreas Schuster and Csaba Barta, takes as an argument Windows filetime date and returns a date in the Epoch format

iso2epoch( string )

Accepts a string that contains a date formatted according to the ISO 8601 and returns an epoch time
