NAME

cftl - An output module for the CFTL XML document

DTD

The DTD for the XML format is the following:

<!ELEMENT EvidenceCollection (Evidence*)>

<!ELEMENT Evidence (Chunk+,Timestamp*,Data*)> <!ATTLIST Evidence title CDATA #REQUIRED> <!ATTLIST Evidence type CDATA #REQUIRED> <!ATTLIST Evidence id CDATA #REQUIRED> <!ATTLIST Evidence parent CDATA "">

<!ELEMENT Chunk EMPTY> <!ATTLIST Chunk from CDATA #REQUIRED> <!ATTLIST Chunk to CDATA #REQUIRED>

<!ELEMENT Timestamp EMPTY> <!ATTLIST Timestamp type CDATA #REQUIRED> <!ATTLIST Timestamp value CDATA #REQUIRED> <!ATTLIST Timestamp origin CDATA #REQUIRED>

<!ELEMENT Data EMPTY> <!ATTLIST Data name CDATA #REQUIRED> <!ATTLIST Data value CDATA #REQUIRED>
